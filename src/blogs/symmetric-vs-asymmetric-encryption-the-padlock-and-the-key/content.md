# Symmetric vs Asymmetric Encryption: The Padlock and the Key

**Difficulty:** Intermediate | **Domain:** Cybersecurity

> *How data is mathematically scrambled to keep it safe from Man-in-the-Middle attacks.*

### The Karnataka Context

Symmetric is using one key for a lock. Asymmetric is having an open padlock anyone can snap shut (Public Key), but only you have the physical key to open it (Private Key).
