# TCP vs UDP: Registered Post vs Putting up a Poster

**Difficulty:** Intermediate | **Domain:** Computer Networks (CN)

> *Reliable ordered delivery versus fast but unreliable broadcasting.*

### The Karnataka Context

TCP is sending a registered speed post acknowledging receipt. UDP is sticking a missing dog poster on an electric pole in Jayanagar hoping someone sees it.
