# REST vs GraphQL: Ordering a Thali vs Custom Dosa

**Difficulty:** Intermediate | **Domain:** Web Development

> *Two different API design paradigms and when to use them.*

### The Karnataka Context

REST is ordering a South Indian Thali (you get what's fixed). GraphQL is customizing exactly how much ghee and chutney you want on your dosa.
