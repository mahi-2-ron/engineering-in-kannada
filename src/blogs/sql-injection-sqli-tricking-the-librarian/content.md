# SQL Injection (SQLi): Tricking the Librarian

**Difficulty:** Intermediate | **Domain:** Cybersecurity

> *Exploiting poorly sanitized database inputs to dump or delete databases.*

### The Karnataka Context

Writing "John; DROP TABLE students;" on an admission form so when the clerk types it exactly into their system, it deletes the entire college database.
