# Edge Computing: The Local Traffic Cop

**Difficulty:** Advanced | **Domain:** Emerging Technologies

> *Processing data locally on devices rather than sending it all the way to a distant cloud server.*

### The Karnataka Context

Instead of sending all camera footage of a traffic jam to a central server in Mumbai to decide if the light should turn green, a local computer at the junction makes the decision instantly.
